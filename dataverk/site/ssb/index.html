Fylker utdanningsnivå

Metoder for å hente og oppdatere datasett basert på SSB tabell 08921: Utdanningsnivå pr fylke

Hent datasett fra cache 

get_from_cache(format='pandas', cache='dvh')

cache: 'dvh' | 'google storage' | 's3'

Hent datasett fra kilde 

get_from_source(format='pandas')

format: 'none' | 'json' | 'pandas' | 'arrow'

eksempel: 

import pandas as pd
from data.datasets.ssb.fylker_utdanningsnivå import FylkerUtdanning

df = FylkerUtdanning.get_from_source(format='pandas')

Oppdater datasett i cache 

update_cache(format='pandas', cache='dvh')

cache: 'dvh' | 'google storage' | 's3' | 'plasma'
